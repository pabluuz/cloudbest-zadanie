# Binary Tree Algorithm

## Setup:
requirements:
- php 7.2 with enabled sqlite
- composer

steps:
1. run composer
- ```composer install```

2. get database
- ```mv app-copy-to-var.db var/app.db```  
  OR
- ```php bin/console do:da:cr```
- ```php bin/console do:mi:mi```
- ```php bin/console do:fi:lo```
